### Install


### Develop
This project is written in coffescript, use the default `gulp` task to start watching `./src/` and compiling to `./lib/`:
```shell
$ git clone https://github.com/learn-co/nsync-fs.git
$ cd nsync-fs
$ npm install
$ npm link
$ gulp
```
